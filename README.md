🎮 Simon Game Web App
A fun and interactive memory-based game where players repeat an increasingly complex sequence of colors and sounds. Test your memory and see how far you can go!

🚀 Features
🧠 Memory-based gameplay (classic Simon game logic)
🎨 Four colored buttons: green, red, yellow, blue
🔊 Sound effects for each button
⌨️ Start game with keyboard input
📈 Progressive difficulty (sequence gets longer each level)
💥 Game-over feedback with restart option

🛠️ Technologies Used
HTML5
CSS3
JavaScript
jQuery

🎮 How to Play
Press any key to start the game.
Watch the pattern of colors that lights up.
Repeat the sequence by clicking the buttons.
Each level adds a new color to the sequence.
If you click the wrong color → ❌ Game Over.
